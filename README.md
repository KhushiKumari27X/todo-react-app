# To-Do React App

A simple and responsive To-Do application built using React. This project allows users to add, delete, and manage daily tasks efficiently, with data stored in the browser using local storage.

## Overview

This application provides a clean interface for managing tasks. It demonstrates core React concepts such as components, props, state management, and local storage handling.

## Features

- Add new tasks  
- Delete existing tasks  
- Persistent data using localStorage  
- Simple and user-friendly interface  
- Real-time updates without page reload  

## Technologies Used

- React.js  
- JavaScript (ES6)  
- CSS  

## Project Structure

- `src/` → Contains all React components  
- `public/` → Static files and HTML  
- `App.js` → Main application logic  
- `components/` → Individual UI components  

## How to Run Locally

1. Clone the repository  
```bash
git clone https://github.com/KhushiKumari27X/todo-react-app.git
```

2. Navigate to the folder  
```bash
cd todo-react-app
```

3. Install dependencies  
```bash
npm install
```

4. Start the app  
```bash
npm start
```
