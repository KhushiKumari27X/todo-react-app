#  To-Do React App

A simple and responsive **To-Do application** built using React. This project allows users to efficiently manage daily tasks with persistent storage using the browser's local storage.

##  Overview

This application provides a clean and intuitive interface for managing tasks. It demonstrates core React concepts such as **components, props, state management, and local storage handling**.

---

##  Features

-  Add new tasks  
-  Delete existing tasks  
-  Persistent data using localStorage  
-  Real-time updates without page reload  
-  Responsive and user-friendly UI  

---

##  Technologies Used

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3
